- 查看所有数据库

```
show databases;
```

- 选择数据库

```
use 数据库名称;
```

- 查看当前所在数据库

```
select database();
```

- 清除当前屏幕命令

```
mysql> \! clear
```

- 查看数据表

```
show tables;
```

- 查看表的字段

```
DESC table_name;
```

中文

```
character set gbk,
```

- 退出 MySQL

```
exit;
```

- 刷新 MySQL

```
mysql> flush privileges;
```

- 查看当前登录用户

```sql
SELECT USER();
```
